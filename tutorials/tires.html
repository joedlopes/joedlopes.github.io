What is Linear Regression?

Linear regression is a simple, powerful method to model the relationship between an input \(x\) and an output \(y\) using a straight line. We assume: \[ \text{Predicted } y = f(x) = w\,x + b \] and choose \(w\) and \(b\) so the line fits the data “as well as possible.”

The Goal

Our objective is to fit the best straight line through the data: minimize the mismatch between actual \(y\) values and the line’s predictions \(f(x)\).

The Model: A Single Neuron

We can view linear regression as a one-neuron model: \[ f(x) = w\,x + b \]

  • \(w\) (Weight): controls the slope (how fast \(y\) changes with \(x\)).
  • \(b\) (Bias): shifts the line up/down.

Measuring Error

For each point \((x_i, y_i)\), the prediction is \( \hat{y}_i = f(x_i) = w x_i + b \). The point-wise error is: \[ e_i = y_i - \hat{y}_i \quad\text{(Actual − Predicted)} \] Residual lines visualize \(e_i\) as vertical distances from points to the model line.

The Loss Function: MSE

We aggregate errors with the Mean Squared Error:

\[ \mathrm{MSE}(w,b) \;=\; \frac{1}{N}\sum_{i=1}^{N}\bigl(y_i - (w x_i + b)\bigr)^{2} \]

  • Mean: average across all \(N\) points.
  • Squared: penalizes large errors more strongly.
  • Error: the residual \(y_i - \hat{y}_i\).

Summary: Goal Revisited

Training linear regression means finding \(w\) and \(b\) that minimize MSE. In the F1 tire example, a good fit estimates how lap times degrade with laps; teams can compare MSE across stints to judge whether the tire has degraded enough to pit.

Next Steps: Gradient Descent

How do we actually find the best \(w\) and \(b\)? A common choice is Gradient Descent, which iteratively updates parameters in the direction that reduces MSE the most. This general idea powers many machine-learning models beyond linear regression.
